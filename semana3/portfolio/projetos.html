• Projeto Seletivo Labenu

Em maio de 2021,participei do processo seletivo da escola de programação Labenu, no projeto acima. Neste projeto, foi solicitado para realizar um layout do site idêntico a foto que eles disponibilizaram. Para fazer o layout, utilizei HTML para fazer o corpo, CSS para estilizar, e Javascript no formulário do usuário. Foi um desafio realizar o processo seletivo,e foi uma etapa muito importante na minha vida. Para construir o que foi pedido, passei dias pesquisando e estudando sobre a estrutura básica de HTML, o meu código possuía mais erros do que acertos, mas com muita persistência, consegui entregar o que foi solicitado. Desde o processo seletivo, tive admiração pela Labenu, pois ela me motivou a pesquisar e estudar. Me ensinou a ser mais proativa. Clique para conhecer o site.
